HTML Notes

Visual Studio Code (VSC) Hot Keys

  • CTRL + B => ShowHide Sidebar of VSC
  • CTRL + C => Copy
  • CTRL + V => Paste
  • CTRL + X => Cut
  • CTRL + S = Save
